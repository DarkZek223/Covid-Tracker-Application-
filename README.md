# COVID-19 Tracker Application

A web application developed using Spring Boot that displays global COVID-19 statistics by fetching and processing real-time data from external CSV datasets. The project uses Thymeleaf for dynamic web pages and Apache Commons CSV for parsing data.

## Technologies Used

* Java 16
* Spring Boot 2.5.3
* Spring MVC
* Thymeleaf
* Apache Commons CSV
* Maven

## Features

* Displays global COVID-19 cases
* Fetches data from an external source
* Parses CSV data efficiently
* Dynamic server-side rendering
* Clean and responsive user interface

## Learning Outcomes

* Spring Boot application development
* REST data retrieval
* CSV data parsing
* MVC architecture
* Maven dependency management
